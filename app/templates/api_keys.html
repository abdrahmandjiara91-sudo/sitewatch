API Documentation

Base URL: https://your-domain/v1/

Get All Sites

Returns all your monitored sites and their status.

GET /v1/status
Authorization: Bearer sk_your_api_key_here

Response:

{
  "user": "your_username",
  "plan": "free",
  "sites_limit": 5,
  "sites_count": 2,
  "sites": [
    {
      "name": "Google",
      "url": "https://google.com",
      "is_up": true,
      "response_ms": 120.5,
      "ssl_days_left": 45
    }
  ]
}

Check a Specific Site

Triggers an immediate check and returns the result.

GET /v1/check/https://google.com
Authorization: Bearer sk_your_api_key_here

Response:

{
  "site": "Google",
  "url": "https://google.com",
  "is_up": true,
  "status_code": 200,
  "response_ms": 98.3,
  "ssl_valid": true,
  "ssl_days_left": 45,
  "error": null
}

Rate Limits

  • /v1/status - 60 requests per minute
  • /v1/check/* - 30 requests per minute
